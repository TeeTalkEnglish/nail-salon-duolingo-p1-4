📖 Key Vocabulary

Words you'll learn in this lesson

💅 Nail Words to Know

Round / Oval / Square
Hình dạng móng
"What shape do you want?"
Almond / Coffin / Stiletto
Dạng móng đặc biệt
"I want almond shape."
Squoval
Vuông + Oval kết hợp
"Squoval is square + oval."
Red / Pink / Nude / White
Màu sắc
"I like the pink color."
Glossy / Matte
Bóng / Mờ
"Do you want glossy or matte?"
Glitter / Chrome / Ombre
Kim tuyến / Gương / Chuyển màu
"Chrome nails are shiny."
Short / Medium / Long
Ngắn / Vừa / Dài
"How long do you want?"
Pretty / Beautiful / Elegant
Đẹp / Xinh / Sang trọng
"Your nails look beautiful!"
💬 Sample Conversation

This is the kind of conversation you'll practice

At the Nail Salon

💅
Nail Tech What shape would you like?
👩
Client I want almond shape, please.
💅
Nail Tech Great! What color do you like?
👩
Client I like light pink.
💅
Nail Tech Do you want glossy or matte finish?
👩
Client Glossy, please!
💅
Nail Tech Perfect! That will look beautiful!
